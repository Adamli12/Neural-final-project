# Biological_Learning
Example of "biological" learning for MNIST based on the paper [Unsupervised Learning by Competing Hidden Units](https://arxiv.org/abs/1806.10181) by D.Krotov and J.Hopfield. 

## Getting started

install jupyter notebook and numpy, scipy, matplotlib.

```bash
> jupyter notebook
```
run `Unsupervised_learning_algorithm_MNIST.ipynb` and observe weights.
